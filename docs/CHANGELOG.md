## [1.0.0] - 2026-01-28

### Добавлено
- Инициализация проекта AI Client Report Generator
- Реализован CLI интерфейс с аргументами командной строки
- Интеграция с OpenAI API для анализа диалогов
- Система кэширования AI-ответов на основе SHA256 хеша
- Генерация PDF отчётов через WeasyPrint
- HTML/CSS шаблоны для отчётов с адаптивным дизайном
- Pydantic схемы для строгой валидации данных
- Система логирования (консоль + файл)
- Retry логика для обработки невалидных JSON ответов от AI
- Обработка ошибок с понятными сообщениями пользователю
- Пример транскрипта диалога в fixtures/
- Подробная документация в README.md

### Структура данных отчёта
- client_name: Имя клиента
- topic: Тема обращения
- main_request: Основной запрос
- sentiment: Анализ настроения (label + score 1-5)
- summary: Краткое содержание
- key_points: Ключевые моменты (список)
- next_steps: Рекомендуемые действия (список)

### Возможности CLI
- `--input`: Указание файла транскрипта
- `--output`: Указание пути к выходному PDF
- `--template`: Выбор HTML шаблона
- `--use-cache / --no-cache`: Управление кэшированием
- `--log-level`: Уровень логирования

### Технологии
- Python 3.8+
- OpenAI API (официальный SDK)
- Jinja2 для шаблонизации
- WeasyPrint для PDF
- Pydantic для валидации
- python-dotenv для конфигурации

- Всего файлов: 23 (без __pycache__)
